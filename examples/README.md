# Examples

This directory contains representative outputs for the major Continue Enterprise Engineering Pack workflows.

The examples are intentionally small. They demonstrate expected structure, tone, and level of detail rather than exhaustive analysis.

## Available Examples

- `repository-discovery.md`
- `implementation-plan.md`
- `code-review.md`
- `architecture-review.md`
- `security-review.md`
- `performance-review.md`
- `release-readiness.md`

## Usage

Use these examples to compare prompt output quality during validation:

1. Run the matching prompt in Continue.
2. Compare the generated structure with the example.
3. Check whether the response separates evidence, assumptions, risks, and next steps.
4. Update the prompt or rule only when the difference reveals a reusable quality issue.
